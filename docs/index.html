홈 · 스마트 검색

검색카테고리혜택

지역·서비스·평점으로 내 주변 관리사를 빠르게 찾습니다. 마사지부터 귀청소·왁싱까지 홈케어 카테고리로 확장.

  • 지역 · 서비스 · 가격 · 평점 필터
  • 카테고리(스웨디시·아로마·귀청소·왁싱 등)
  • 혜택 · 쿠폰 · 이벤트 배너

지금 바로 예약

온디맨드취향 매칭

강도·집중 부위·시간 취향만 고르면 가까운 관리사를 5분 안에 자동 매칭해 드립니다.

  • 강도(약·보통·강) · 집중 부위 선택
  • 거리 기반 즉시 매칭
  • 예상 도착 시간 안내

  • 신원인증 · 건강검진 · 안심결제 배지
  • 서비스 · 가격 · 실시간 예약 가능 여부
  • 실제 고객 리뷰 · 평점

예약 · 달력

캘린더단계형

서비스 → 날짜 → 시간 → 주소, 직관적인 달력 UI로 몇 번의 탭이면 예약 완료.

  • 서비스 · 소요시간 선택
  • 달력에서 날짜 · 시간 선택
  • 방문 주소 입력

예약 확정 · 결제

국내 PG 연동옵션 선택형

예약 내역과 금액을 투명하게 확인하고 결제합니다. 국내 PG사 1종을 연동하며, 아래 결제 옵션 중 선택해 적용합니다.

  • 국내 PG사 연동 (1종 · 견적 기준)
  • 옵션: 신용·체크카드 / 간편결제(카카오·네이버페이 등) / 계좌이체
  • Wallet(선불 충전 · 포인트) 연동
  • 최종 결제 수단은 협의 후 선택

양방향 평점 시스템

별점매너 평점

서비스 후 별점·태그·후기를 남기고, 고객도 관리사로부터 매너 평점을 받는 양방향 신뢰 구조.

  • 별점 · 빠른 태그 · 상세 후기
  • 고객 매너 평점(받은 평가) 조회
  • 서로 존중하는 이용 문화

지도거리순

실시간 지도에서 가까운 관리사를 거리순으로 확인하고 바로 예약할 수 있습니다.

  • 지도 기반 주변 탐색
  • 가까운 순 정렬
  • 실시간 위치(연동 예정)

  • 원터치 SOS · 실시간 위치 공유
  • 112 긴급 연결

관리자 웹 · Admin

플랫폼을 안전하게 운영하는 통합 포털

회원·정산 관리와 함께, 신고와 블랙리스트를 연동한 신뢰·안전 운영 도구를 갖췄습니다.

신고 관리 시스템

고객↔테크니션 양방향 신고를 한 곳에서 접수·검토·처리합니다. 상태별 필터, 처리중/완료/반려 처리, 그리고 신고 → 블랙리스트 등록까지 실제로 연동됩니다.

  • 양방향 신고 큐 · 상태 필터
  • 처리중 · 처리완료 · 반려
  • 신고 → 블랙리스트 원클릭 등록

블랙리스트 관리

반복 노쇼·불량 이용자를 등록·해제하고, 신고 건과 연동해 관리합니다. 수동 등록도 지원해 운영 정책을 유연하게 적용할 수 있습니다.

  • 고객 · 테크니션 이용 제한 관리
  • 활성 / 해제 토글 · 신고 연동
  • 관리자 수동 등록
